# `serial stopbits <bits>`

---

## Function

Configures the number of stop bits (either 1 or 2).

## Example
```
[admin@ssh-to-serial]>serial stopbits 2
UART Config - Base: 0x4000D000
 Clock: 120000000 Hz
 Baud: 115200
 Data Size: 8
 Parity: N
 Stop Bits: 2
 Flow Control: N
 Transmit Delay: 300 ms
Serial stop bits set to 2.
```
